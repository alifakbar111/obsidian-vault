This shows the system correctly values **depth of focus** alongside **length of career**.

## Key Observations from This Test

### ✅ What worked well

1. **Education bonus correctly tiered** — Master's (+45) vs Bachelor's (+30). The Master's adds meaningful value but doesn't overshadow experience.

2. **Tool vs concept distinction** — PHP (+0 edu), FastAPI (+0 edu), ElasticSearch (+0 edu) correctly got no education bonus since they're learned on the job, not in university. Python and SQL (+45 edu each) correctly got the bonus.

3. **Certifications properly scoped** — HackerRank Python cert added +10 to Python. Udacity Data Foundations added +10 to ML. Each cert is scored individually rather than a single boolean.

4. **Overlapping roles handled** — The Apprentice Mentor (Sep-Nov 2021) overlapped with Widya Data Scientist (Jan-Nov 2021). Calendar years for Python don't double-count this period.

### ❓ Edge cases to watch

1. **"Data Team Lead" → Senior (+65) or Expert (+100)?** Our table says "Lead" → Senior. But "Data Team Lead" supervised data scientists, engineers, and DBAs — team-level scope. The description doesn't mention Director/VP/C-level. I think Senior (+65) is correct.

2. **"IT Manager" → Senior (+65) or Expert (+100)?** Title says "Manager" but our classification says "Manager → Expert with caveat (scope)". The role was 4 months at a freelance/small company, advising a CEO. Is that Expert (org-wide)? Or Senior (team-level)? I classified as Senior (+65) since it's a short-term freelance role.

3. **Teaching Assistant role weight** — TA is professional work but not senior. Should it be "Skilled" (+35)? Or "Entry" (+10)? I gave it Skilled because teaching at Master's level requires expertise. But reasonable people could disagree.
