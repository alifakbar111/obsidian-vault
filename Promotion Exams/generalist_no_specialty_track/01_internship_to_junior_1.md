# Promotion Exam 01 — Internship → Junior Engineer 1

**Format:** 100 multiple-choice questions, one correct answer each.
**Time limit (suggested):** 120 minutes.
**Passing score:** 89 / 100.
**Scope:** Programming fundamentals, version control, web basics, simple algorithms and data structures, basic SQL, basic CLI, debugging mindset. This exam is meant to confirm that an intern has the *foundations* needed to be trusted with small tickets under supervision.
