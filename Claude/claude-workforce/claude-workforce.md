# Your Claude Workforce

**42 real skills. 7 crews. One workspace.**

A "Claude Workforce" is the idea of treating Claude Code not as a single assistant but as a whole company: one `claude-code` workspace staffed by **7 departments (crews)**, each holding **6 installable skills** — 42 in total. Every skill is a real, installable Claude Code skill that gives Claude a specialized capability, from writing tests to building financial models to reviewing contracts. Instead of one generalist, you get a full org chart of specialists you can call on.

## 01 / Developers

*Ship code faster, from scaffold to QA.* — **Your build team.**

### Superpowers
*Full skill pack for planning + TDD.*
A bundled toolkit that gives Claude a disciplined engineering workflow: brainstorm the design, write a spec, plan the implementation, then build test-first. It front-loads thinking so you don't get half-baked code, and enforces test-driven development so behavior is verified as it's written. Reach for it when a feature is big enough that jumping straight to code would waste effort.

### Context7
*Pulls live, version-exact docs.*
Fetches up-to-date documentation for the exact version of a library you're using, instead of relying on the model's training snapshot. This kills the classic failure mode where Claude suggests an API that was renamed or removed two releases ago. Use it any time you're integrating a fast-moving framework or an unfamiliar package.

### MCP Builder
*Wire Claude to any tool.*
Scaffolds a Model Context Protocol server so Claude can talk to an external system — a database, an internal API, a SaaS product — as a first-class tool. It turns "Claude can't reach that" into "Claude can now read and act on it." Use it when you want Claude to operate your own tools, not just the ones that ship out of the box.

### Skill Creator
*Scaffold your own skills.*
Generates the structure of a new custom skill — the frontmatter, the instructions, the trigger description — so you can codify a repeatable workflow of your own. It's the meta-skill that lets your workforce grow. Use it when you find yourself explaining the same procedure to Claude more than once.

### Webapp Testing
*Browser-tests your app.*
Drives a real browser to click through your running web app, fill forms, and verify that flows actually work end to end. It catches the bugs unit tests miss — broken buttons, dead routes, JS errors in the console. Use it after shipping a UI change to confirm the happy path still holds.

### Claude-Mem
*Memory across sessions.*
Persists facts, decisions, and context so Claude remembers your project between conversations instead of starting cold each time. It means you stop re-explaining your architecture and preferences every morning. Use it on any long-running project where continuity matters.

## 02 / Design

*UI that never looks templated.* — **Your design studio.**

### frontend-design
*Bold React + Tailwind UI.*
Produces distinctive, intentional interfaces in React and Tailwind rather than the generic "default admin dashboard" look. It focuses on deliberate typography, spacing, and aesthetic direction. Use it when you want a UI that feels designed, not scaffolded.

### web-artifacts
*shadcn HTML artifacts.*
Builds self-contained web components and pages using the shadcn/ui design language, delivered as shareable artifacts. Good for polished, accessible building blocks without reinventing component styling. Use it to spin up clean, on-brand UI pieces quickly.

### canvas-design
*Visual art to PNG / PDF.*
Turns visual concepts into finished, exportable graphics — posters, diagrams, layouts — rendered out to PNG or PDF. It bridges "describe the visual" and "hand me the file." Use it for one-off graphics you need as real image or document assets.

### algorithmic-art
*Generative p5.js art.*
Creates generative, code-driven artwork with p5.js — patterns, animations, and visuals produced by algorithms rather than drawn by hand. Great for unique backgrounds, generative branding, or creative experiments. Use it when you want art that's parametric and endlessly variable.

### ui-ux-pro-max
*Full design-system intel.*
A deep design-system reference covering styles, color palettes, font pairings, layout, accessibility, and component patterns. It acts as an opinionated design consultant to guide or audit UI decisions. Use it when planning a coherent visual system or reviewing UI code for consistency.

### slack-gif
*Slack-ready animated GIFs.*
Generates short animated GIFs sized and formatted for Slack — reactions, celebrations, status visuals. A small but delightful team-culture tool. Use it to make internal comms more fun without opening a design app.

## 03 / Marketing

*Copy, SEO and ads that convert.* — **Your growth engine.**

### seo-audit
*Diagnoses on-page SEO.*
Analyzes a page's on-page SEO — titles, meta, headings, structure, internal links — and reports what's hurting your rankings. It hands you a prioritized fix list instead of vague advice. Use it as a first pass before investing in content or link building.

### programmatic-seo
*Pages at scale from data.*
Generates large sets of SEO-optimized pages from a structured dataset — think "best X in every city" templates populated automatically. It lets you target thousands of long-tail queries without writing each page by hand. Use it when you have data and a repeatable page pattern.

### ai-seo
*Rank inside AI answers.*
Optimizes content so it gets cited inside AI-generated answers (ChatGPT, Claude, AI Overviews), not just classic search results. As search shifts to AI summaries, this protects your visibility. Use it to make sure your brand shows up where people increasingly look.

### cro
*Lift page + form conversion.*
Focuses on conversion-rate optimization: improving the copy, layout, and forms so more visitors actually take action. It targets the gap between traffic and revenue. Use it when you have visitors but not enough sign-ups or sales.

### ad-creative
*Scales ad variations.*
Rapidly produces many variations of ad copy and creative concepts for testing across platforms. It feeds the volume that paid-ad testing demands. Use it when you need dozens of angles to find the winners.

### mktg-psychology
*Behavioral triggers.*
Applies behavioral-psychology principles — social proof, scarcity, anchoring, loss aversion — to messaging and offers. It makes copy persuasive on purpose rather than by luck. Use it to sharpen landing pages, pricing, and CTAs.

## 04 / Social & Content

*Feed the algorithm on autopilot.* — **Your content machine.**

### social
*Posts for every platform.*
Writes platform-native posts tuned to the format and tone of each network (X, LinkedIn, Instagram, etc.) from a single idea. It saves you from manually reformatting one message seven ways. Use it to keep a consistent presence across channels.

### copywriting
*Rewrites any page copy.*
Rewrites and sharpens existing copy — landing pages, product descriptions, emails — for clarity and persuasion. It's the editor that tightens what you already have. Use it when the message is right but the words aren't landing.

### content-strategy
*Plans your topic map.*
Builds a topic map and content plan: which subjects to cover, how they cluster, and what order to publish. It gives your content a coherent structure instead of scattered one-offs. Use it at the start of a content push to set direction.

### video
*Scripts + produces video.*
Writes video scripts and helps produce the finished piece — hooks, structure, shot notes, captions. It turns an idea into a shootable, publishable video. Use it for short-form social video or explainer content.

### pillar-content
*Hub-and-cluster authority.*
Creates comprehensive "pillar" pages surrounded by supporting cluster articles, the hub-and-spoke model that signals topical authority to search engines. It's a long-game SEO structure. Use it to dominate a topic rather than rank for a single keyword.

### email-sequences
*Lifecycle email flows.*
Designs automated email sequences for each stage of the customer lifecycle — welcome, onboarding, nurture, win-back. It keeps engagement running without manual sends. Use it to convert and retain subscribers on autopilot.

## 05 / Finance

*Model the numbers before you spend.* — **Your CFO on call.**

### dcf-model
*Discounted cash-flow value.*
Builds a discounted-cash-flow model to estimate the intrinsic value of a business or project by projecting future cash flows and discounting them to today. It answers "what is this actually worth?" Use it for valuation, investment decisions, or sanity-checking a price.

### 3-statements
*Full 3-statement model.*
Constructs a linked income statement, balance sheet, and cash-flow statement that flow into each other. It's the backbone financial model every other analysis builds on. Use it to model a company's full financial picture over time.

### lbo-model
*Leveraged-buyout math.*
Models a leveraged buyout — the debt structure, returns, and cash flows of acquiring a company primarily with borrowed money. It shows whether a deal generates the target return. Use it for private-equity-style acquisition analysis.

### comps-analysis
*Comparable-company set.*
Assembles a set of comparable companies and their valuation multiples to benchmark a target relative to its peers. It's the market-based counterpart to a DCF. Use it to cross-check valuation against what similar businesses trade for.

### pricing
*Packaging + monetization.*
Designs pricing tiers, packaging, and monetization strategy — how to structure and charge for a product. It connects value delivered to revenue captured. Use it when launching or repricing a product or subscription.

### pitch-deck
*Investor pitch decks.*
Builds investor-ready pitch decks: narrative, market, traction, financials, and ask, in a clear slide flow. It turns your story into something fundable. Use it when raising capital or presenting to stakeholders.

## 06 / Operations

*Run the business like a machine.* — **Your ops backbone.**

### sop-builder
*Writes clean SOPs.*
Documents standard operating procedures — clear, step-by-step processes anyone on the team can follow. It captures know-how so it doesn't live only in someone's head. Use it to standardize recurring work and onboard people faster.

### incident-postmortem
*Blameless post-mortems.*
Produces blameless post-mortems after incidents: timeline, root cause, impact, and action items, without finger-pointing. It turns failures into systemic fixes. Use it after any outage or major mistake to prevent a repeat.

### business-case
*ROI + business cases.*
Builds structured business cases with ROI analysis to justify a decision or investment. It frames costs, benefits, and payback so stakeholders can decide. Use it to get buy-in for a project or spend.

### launch-runbook
*Go-live + DNS cutover.*
Creates a detailed launch runbook covering go-live steps, DNS cutover, rollback plans, and checks. It makes launches boring and predictable instead of stressful. Use it before shipping a product, site, or migration to production.

### internal-comms
*Status reports + FAQs.*
Writes internal communications — status updates, announcements, and FAQ documents — that keep the team informed and aligned. It reduces the "what's going on?" overhead. Use it for recurring updates and change communications.

### xlsx
*Excel with live formulas.*
Generates real Excel spreadsheets with working formulas, not static tables — so the numbers recalculate when inputs change. It delivers files people can actually use and edit. Use it whenever the output needs to be a functioning spreadsheet.

## 07 / Legal

*Read the fine print for you.* — **Your legal desk.**

### contract-review
*Clause-by-clause review.*
Reviews contracts clause by clause, flagging risky, unusual, or unfavorable terms in plain language. It gives you a first-pass read before a lawyer's time. Use it to understand what you're signing and where the traps are.

### nda-triage
*Auto-triage NDAs.*
Automatically triages non-disclosure agreements — classifying them as standard, needs-attention, or problematic — so routine ones move fast. It clears the backlog of low-risk paperwork. Use it when you sign NDAs often and want to prioritize the ones that matter.

### legal-risk
*Flags legal exposure.*
Scans documents and situations for legal exposure and highlights where you might be at risk. It's an early-warning layer, not a substitute for counsel. Use it to spot issues before they become problems.

### compliance
*Regulatory checks.*
Checks documents, processes, or products against relevant regulatory requirements. It helps you stay on the right side of the rules. Use it when operating in a regulated space or preparing for an audit.

### docx
*Word docs, tracked changes.*
Produces and edits Microsoft Word documents with proper formatting and tracked changes — the format law and business run on. It fits the redline workflow lawyers actually use. Use it when the deliverable must be an editable, review-ready .docx.

### sql-queries
*Pull records with SQL.*
Writes and runs SQL to pull specific records from a database — useful for discovery, audits, and evidence gathering. It gets the exact data a legal or compliance question needs. Use it when the answer lives in your database.
